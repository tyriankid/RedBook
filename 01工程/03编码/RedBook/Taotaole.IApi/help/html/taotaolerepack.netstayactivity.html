方法名称: staysentredpack (留存活动触发发包动作接口)
请求示例
domain测试地址为：http://api.ewaywin.com
domain/?/ilerrredpack?action=staysentredpack&sign=MW_B5ECA031A5D9F85F2972E648AA8FA648&shopid=1
参数说明
参数名称 类型 是否必须字段 说明
sign STRING true 用户ID随机签名，不接收未知的签名
uid STRING true 用户ID随机签名，不接收未知的签名
shopid STRING true 一元购商品ID
返回结果
{
state:"1", //个人私有数据凭证状态，0执行完毕 1未获得业务ID 2未知的签名参数
}
