方法名称: myconsume (我的消费记录)
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerrpay?action=myconsume&sign=用户ID随机签名&p=页码
参数说明
参数名称 类型 是否必须字段 说明
sign STRING true 用户ID随机签名，不接收未知的签名
p STRING false 页码
返回结果
{
state:"0", //个人私有数据凭证状态，0验证通过 1缺少签名参数 2未知的签名参数
count："2", //实际消费的总数量
pageCount:"3", //总共页数
nextPage:"0", //下一页页码
data:[] //返回数据列表
{
content:'云购了商品',//标题--title
time:'',//交易时间
money:'0.01',//充值金额
}
