方法名称: updateuserinfo (更改用户信息（昵称，签名）)
方法名称: updateuserimg (更改用户信息（头像）)
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerruser?action=updateuserinfo&sign=用户ID随机签名&username=用户昵称&qianming=用户签名
请求示例（更改用户信息（头像））
domain测试地址为：http://platform.ewaywin.com
domain/ilerrshareorder?action=updateuserimg&sign=用户ID随机签名&file1=用户头像（base64位编码）
参数说明
参数名称 类型 是否必须字段 说明
sign STRING true 用户ID随机签名，不接收未知的签名，微信不用传此参数
username STRING false 用户昵称
qianming STRING false 用户昵称
参数说明（游戏）
参数名称 类型 是否必须字段 说明
sign STRING true 用户ID随机签名，不接收未知的签名，微信不用传此参数
file1 STRING true 用户头像（base64位编码）
返回结果
{
state:"0", //设置默认收货地址，0更新成功 //2未知的签名参数
}
更改头像：{
state:"0", //设置默认收货地址，0设置成功 //2未知的签名参数 //3头像base64编码不存在
}
