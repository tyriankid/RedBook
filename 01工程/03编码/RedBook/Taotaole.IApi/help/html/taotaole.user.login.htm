方法名称: login (用户登录)
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerruser?action=login&username=用户名&password=密码
参数说明
参数名称 类型 是否必须字段 说明
username string true 用户名
password string true 密码，需通过base64encode进行编码
返回结果
{
state:"0", //登录验证状态，0验证通过 2帐号或密码错误 3未通过验证
sign:"1d08BwhVBlUACFQ", //用户ID随机签名，获取个人私有数据的凭证
}
