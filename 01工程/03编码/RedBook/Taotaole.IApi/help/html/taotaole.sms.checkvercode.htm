方法名称: validatecode (验证短信验证码)
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerrsms?action=validatecode&mobile=手机号&code=验证码
参数说明
参数名称 类型 是否必须字段 说明
mobile STRING true 手机号
code STRING true 手机号
返回结果
{
state:"0", //0代表验证成功 2代表验证失败 3代表验证过时，请重新发送验证码 4代表验证错误 5验证码失效，请重新发送验证码
sign:"1d08BwhVBlUACFQ", //用户ID随机签名，获取个人私有数据的凭证
}
