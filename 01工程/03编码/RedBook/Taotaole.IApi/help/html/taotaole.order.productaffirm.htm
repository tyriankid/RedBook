方法名称: productaffirm (商品确认（中奖之后点击的商品确认）)
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerrshareorder?action=productaffirm&sign=用户ID随机签名&orderid=订单ID&shopid=一元购商品ID
参数说明
参数名称 类型 是否必须字段 说明
sign STRING true 用户ID随机签名，不接收未知的签名，微信不用传此参数
orderid STRING true 订单ID
shopid STRING true 一元购商品ID
