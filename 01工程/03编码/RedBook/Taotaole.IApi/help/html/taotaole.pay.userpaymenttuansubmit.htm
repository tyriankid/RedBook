方法名称: userpaymenttuansubmit (用户提交团购付款)
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerrpay?action=userpaymenttuansubmit&sign=用户ID随机签名&typeid=类型ID&businessid=业务ID
参数说明
参数名称 类型 是否必须字段 说明
sign STRING true 用户ID随机签名，不接收未知的签名，微信不用传此参数
typeid STRING true 参团方式(1团长开团、2成员参团)
businessid STRING true 业务ID(团长开团时传团购商品ID，成员参团时传团长ID[团购商品参与主键ID])
返回结果
{
state:"0", //状态(0-生成订单成功 -1-请不要重复提交订单 1-用户不存在 2-团购已过期 3-团购数已满 4-余额不足 5-已参过此团 6-提交订单失败,请稍后重试)
orderid："Y163" //团购订单号
}
