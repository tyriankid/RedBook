方法名称: redpacklist (获取购物时可用红包列表)
请求示例
domain测试地址为：http://api.ewaywin.com
/ilerrredpack?action=redpacklist&sign=MW_B5ECA031A5D9F85F2972E648AA8FA648&money=1000&uid=2BD59896A291D2F9
参数说明
参数名称 类型 是否必须字段 说明
action STRING true 方法名
sign STRING true 随机签名，不接收未知的签名
uid STRING true 用户ID随机签名，不接收未知的签名
money STRING true 消费金额
返回结果
{
state:"0", //个人私有数据凭证状态，0验证通过 1缺少签名参数 2未知的签名参数
data："Array", //可用红包列表
}
Array{
id:"0", //红包ID
codetitle:"0", //红包标题
amount："0", //满足金额
discount："0", //抵扣金额
}
