方法名称: redpackinfo (获取用户红包)
请求示例
domain测试地址为：http://api.ewaywin.com
/ilerrredpack?action=redpackinfo&sign=MW_B5ECA031A5D9F85F2972E648AA8FA648&uid=2BD59896A291D2F9&typeid=1&p=1
参数说明
参数名称 类型 是否必须字段 说明
action STRING true 方法名
sign STRING true 用户ID随机签名，不接收未知的签名
uid STRING true 用户ID随机签名，不接收未知的签名
typeid STRING true 1可用，2待发，3过期和已使用
返回结果
{
state:"0", //个人私有数据凭证状态，0验证通过 1缺少签名参数 2未知的签名参数
count："0", //红包个数
docount："0", //可用红包个数
savecount："0", //待发红包个数
pageCount："0", //页数
nextPage："0", //下页
data{
id:"0", //红包ID
codetitle："0", //红包标题
amount："0", //限制金额
discount："0", //抵扣金额
state:"0", //红包状态
senttime："0", //生效时间
overtime："0", //过期时间
}
