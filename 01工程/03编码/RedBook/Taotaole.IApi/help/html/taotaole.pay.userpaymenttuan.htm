方法名称: userpaymenttuan (用户团购付款校验)
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerrpay?action=userpaymenttuan&sign=用户ID随机签名&typeid=团购类型ID&businessId=团购业务ID
参数说明
参数名称 类型 是否必须字段 说明
sign STRING true 用户ID随机签名，不接收未知的签名，微信不用传此参数
typeid STRING true 参团方式(1团长开团、2成员参团)
businessid STRING true 业务ID(团长开团时传团购商品ID，成员参团时传团长ID[团购商品参与主键ID])
返回结果
{
state:"0", //个人私有数据凭证状态，0验证通过 2未知的签名参数 3缺少参数 4团购已过期 5团购数已满 6已参过此团
}
