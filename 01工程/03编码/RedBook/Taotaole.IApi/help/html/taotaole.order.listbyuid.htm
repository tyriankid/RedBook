方法名称: oneselfpartake (别人的一元购订单(云购记录))
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerrorder?action=otheroneselfpartake&uid=用户ID&typeid=类别码&p=页码
参数说明
参数名称 类型 是否必须字段 说明
uid STRING true 用户ID
typeid int false 类别码，默认为0。0表示参与云购的商品_全部，1表示参与云购的商品_进行中，2表示_参与云购的商品已揭晓
p int false 页码，默认为1
