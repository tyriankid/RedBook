方法名称: updatepass (修改密码)
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerruser?action=updatepass&sign=用户ID随机签名&pass=原密码&code=短信验证码&passnew=新密码
参数说明
参数名称 类型 是否必须字段 说明
sign STRING true 用户ID随机签名，不接收未知的签名，微信不用传此参数
pass STRING true 原密码，需通过base64encode进行编码
code STRING false 短信验证码，用手机找回密码时需要，可不需要原密码
passnew STRING false 原密码，需通过base64encode进行编码,用于原密码验证过后更改密码
返回结果
{
state:"0", //更新密码状态，0代更新成功 //1代表签名验证错误 //2代更新失败 //3代表验证过时，请重新发送验证码 //4代表验证错误
}
