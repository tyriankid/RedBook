方法名称: tuanlist (团购商品列表)
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerrtuan?action=tuanwinnerinfo&tid=17&type=1&p=1
参数说明
参数名称 类型 是否必须字段 说明
type STRING true 该团类型0为及时开奖 1为限时开奖
tid STRING true 当类型为及时开奖时传businessId 为限时开奖时传tuanId
p STRING false 页码，默认为1
