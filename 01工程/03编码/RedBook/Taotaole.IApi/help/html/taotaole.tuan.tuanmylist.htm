方法名称: tuanmylist (我的团购列表)
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerrtuan?action=tuanmylist&sign=&uid=&status=状态(全部、进行中、成功、失败、已开奖)&p=页码
参数说明
参数名称 类型 是否必须字段 说明
action STRING true tuanmylist
sign STRING true 签名
uid STRING true 用户ID加密后
status STRING true 团购状态（0为全部、1为进行中、2为成团、3为失败、4为已开奖）
p STRING false 页码
