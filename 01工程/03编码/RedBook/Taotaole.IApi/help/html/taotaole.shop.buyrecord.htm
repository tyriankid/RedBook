方法名称: buyrecord (获取商品参与记录)
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerrshop?action=buyrecord&shopid=商品ID&p=页码
参数说明
参数名称 类型 是否必须字段 说明
ShopID int true 一元购商品ID
P int false 页码，默认为1
返回结果
{
state："0", //获取数据状态，0成功获取 1未获取到数据或数据已加载完毕 2商品ID错误
"count":12, //总条数，总数量
"pageCount":2, //总共页数
"nextPage":2, //下一页页码
"firsttime":"2016-4-12 12:00:00" //首次参与时间
data: //返回数据值
{
uid:'1',//购买人ID
username:'张三',//购买人名称
img:'1.jpg',//购买人头像
ip:'192.168.1.106',//购买的ip
time:'2016-07-14 13:00:00',//购买时间
quantity:'6',//购买次数
businessId:'2',//业务ID商品ID 一元购Yid
}
}
