方法名称: userpayment (用户付款)
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerrpay?action=userpayment&sign=用户ID随机签名&shopid=商品ID&quantity=购买数量
参数说明
参数名称 类型 是否必须字段 说明
sign STRING true 用户ID随机签名，不接收未知的签名
shopid STRING true 商品ID
quantity STRING true 购买数量
返回结果
{
state:"0", //个人私有数据凭证状态，0验证通过 1缺少签名参数 2未知的签名参数 3缺少参数 4商品不存在 5商品已售罄 6商品已过期
quantity："3", //实际购买数量
paymoney:"3", //实际待支付金额
APIsubmitcode:"163", //支付唯一码
usemoney:"1", //余额是否足够(1是 0否)
useredpaper:"0", //是否有可用红包(是 0否)
money:"163", //账户余额(元)
data: //可用红包列表
{
id:'1',//红包ID
codetitle:'满5减2红包',//红包标题
amount:'5',//满足金额,为0时不限制
discount:'2',//抵扣金额
}
}
