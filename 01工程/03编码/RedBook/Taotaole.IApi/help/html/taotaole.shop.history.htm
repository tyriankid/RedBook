方法名称: history (获取往期揭晓记录)
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerrshop?action=history&productid=商品ID
参数说明
参数名称 类型 是否必须字段 说明
productid int true 商品详细信息ID
p int false 页码，默认为1
返回结果
{
state:"0", //获取数据状态，0成功获取 1未获取到数据或数据已加载完毕 2代表产品id错误

nextPage:"20", //下一页页码
imgroot:"G_UPLOAD_PATH", //图片根目录
data: //获取返回数据
{
yId:'1',//1元购商品唯一标识
productid:'1',//商品id,来源go_products.productid
username:'张三',//中奖人名称

qishu:'2',//商品期数
quantity:'1',//购买次数
q_code:'00000001',//中奖号码

}
}
