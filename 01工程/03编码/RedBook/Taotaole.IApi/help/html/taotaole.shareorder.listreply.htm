方法名称: shaidanreply (获取某晒单评论列表)
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerrshareorder?action=shaidanreply&sign=用户签名&shaidanid=晒单ID&p=页码
参数说明
参数名称 类型 是否必须字段 说明
sign STRING false 用户签名
shaidanid STRING true 晒单ID
p STRING false 页码，默认为1
返回结果
{
state："0", //获取数据状态，0成功获取 2//未知的签名参数 3//晒单ID有误 4//此商品还没有晒单
count:"3", //总记录数
pageCount:"20", //总页数
nextPage:"163", //下一页页码
imgroot:"G_UPLOAD_PATH", //图片根目录
data:$shaidan //晒单记录实体(仅公开部分属性)
{
username:'小明',//晒单回复人
sdhf_content:'历害的三石哥',//晒单回复内容
sdhf_time:'',//晒单回复时间，精确到秒
img:'',//晒单回复人头像
}
