方法名称: shaidandetails (获取晒单详情)
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerrshareorder?action=shaidandetails&sign=用户签名&shaidanid=晒单ID
参数说明
参数名称 类型 是否必须字段 说明
sign STRING true 用户ID随机签名，不接收未知的签名
shaidanid STRING true 晒单ID
返回结果
{
state:"0", //状态，0验证通过 2未知的签名参数 3商品ID有误 4获取晒单列表失败
imgroot:"G_UPLOAD_PATH", //图片根目录
data:$shaidan //晒单记录实体(仅公开部分属性)
{
username:'理想',//晒单人，中奖人
sd_id:'1',//晒单ID
sd_title:'surface pro 4',//晒单标题
sd_ip:'湖北省武汉市,58.19.7.10',//晒单IP
sd_photolist:'1.gpg;2.jpg',//晒单图片,多图以分号分隔
sd_time:'00000',//晒单时间
sd_shopid:'1',//晒单商品ID
sd_zan:'11',//晒单赞数
sd_ping:'12',//晒单评论数
q_uid:'11',//中奖人ID
title:'50元移动充值话费 酷炫场',//商品标题
q_code:'10000003',//中奖码
qishu:'1',//期数
quantity:'12',//购买次数
q_end_time:'2016-08-09 20:23:02.647',//开奖时间
img:'1.jpg',//中奖人头像 ----------头像是完美路径 不需要添加imgroot
praise:"0"//是否点赞，1已点赞 0未点赞或已取消点赞
}
}
