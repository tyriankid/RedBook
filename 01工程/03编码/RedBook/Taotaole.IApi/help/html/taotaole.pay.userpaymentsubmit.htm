方法名称: userpaymentsubmit (用户提交付款)
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerrpay?action=userpaymentsubmit&sign=用户ID随机签名&shopid=商品ID&quantity=购买数量&redpaper=红包ID&APIsubmitcode=支付校验码&paytype=支付方式
参数说明
参数名称 类型 是否必须字段 说明
sign STRING true 用户ID随机签名，不接收未知的签名
shopid STRING true 商品ID
quantity STRING true 购买数量
paytype STRING true 支付方式(0余额支付、1支付宝支付、2微信支付)
APIsubmitcode STRING true 支付校验码
redpaper STRING false 红包ID
返回结果
{
state:"0", //个人私有数据凭证状态，0购买成功 2未知的签名参数 3缺少参数 4商品不存在 5商品已售罄 6商品已过期 7请不要重复提交订单 8用户不存在 9余额不足 10该期已截止 11提交订单失败,请稍后重试
orderid："Y163" //订单号
}
