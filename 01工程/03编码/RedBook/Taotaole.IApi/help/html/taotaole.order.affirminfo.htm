方法名称: affirminfo (提交信息（中奖之后点击提交信息）)
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerrshareorder?action=affirminfo&sign=用户ID随机签名&orderid=订单ID
参数说明
参数名称 类型 是否必须字段 说明
sign STRING true 用户ID随机签名，不接收未知的签名，微信不用传此参数
orderid STRING true 订单ID
post array true POST提交大文本，格式 data:{mobile:手机号,info:用户提交信息 如果是提交卡密，则需要提供：kahao=卡号;kami=卡密;usetype=使用方式1充值话费2充值支付宝3充值爽乐币;usenum=充值号码}
返回结果
{
state:0 //0充值以完成 2未知的签名参数 3订单ID错误 4用户信息有误 5手机号有误 （6获取订单信息有误 7商品库存不足，请稍后再试，或者联系客服宝宝） 10信息已提交 状态6、7针对发送卡密的状态 10针对实物、游戏，更新状态
}
