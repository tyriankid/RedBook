方法名称: mywinninglist (我的中奖记录)
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerrorder?action=mywinninglist&sign=用户签名&p=页码
参数说明
参数名称 类型 是否必须字段 说明
sign STRING true 用户ID随机签名，不接收未知的签名,微信不用传此参数
p STRING false 页码，默认为1
