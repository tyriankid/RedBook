方法名称: useraddresslist (用户人收货地址列表)
方法名称: usergameaddresslist (获取游戏收货地址列表)（游戏）
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerruser?action=useraddresslist&sign=用户ID随机签名

请求示例（游戏）
domain测试地址为：http://api.ewaywin.com
domain/ilerruser?action=usergameaddresslist&sign=用户ID随机签名
参数说明
参数名称 类型 是否必须字段 说明
sign STRING true 用户ID随机签名，不接收未知的签名，微信不用传此参数

参数名称 类型 是否必须字段 说明
