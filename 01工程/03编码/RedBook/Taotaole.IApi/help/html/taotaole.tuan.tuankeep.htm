方法名称: tuankeep (用户收藏团购商品或取消收藏)
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerrtuan?action=tuankeep&type=0收藏/1取消收藏/2获取收藏状态&tId&sign签名
参数说明
参数名称 类型 是否必须字段 说明
type STRING false 操作类型0收藏1取消收藏2获取收藏状态返回state 0为已收藏 1为未收藏
tId STRING false 团Id
sign STRING false 签名APP需传签名
返回结果
{
state："0", //当type=0时 返回2为收藏成功 3为失败 当type=1是返回0为取消成功 1为取消失败 当type=2时返回0为已收藏 1为未收藏
}
