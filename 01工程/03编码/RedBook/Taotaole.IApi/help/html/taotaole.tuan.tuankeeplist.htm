方法名称: tuanlist (用户收藏团购商品列表)
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerrtuan?action=tuankeeplist&p=页码
参数说明
参数名称 类型 是否必须字段 说明
p STRING false 页码，默认为1

"pageCount":2, //总共页数
"nextPage":2, //下一页页码

data: //返回数据值
{
tId:'1',//团购商品ID

total_num:'20',//成团数量

}
}
