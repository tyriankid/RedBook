方法名称: slides (获取幻灯片)
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerrarticle?action=slides&slidetype=幻灯片类型
参数说明
参数名称 类型 是否必须字段 说明
slidetype STRING true 类型：1微信 2APP 3商城
微信和商城的返回结果
{
state："0", //获取数据状态，0成功获取 2缺少参数 3错误的参数
imgroot:"G_UPLOAD_PATH", //图片根目录
data: //返回数据值
{
slideid:50,//幻灯片唯一标识id
slidelink:'www.baidu.com',//链接地址(当类型slidetype=2时查询的是app数据量过大 不显示)
slideimg:'slides/20160817/products_090542.jpg',//幻灯片地址
slidetype:1,//幻灯片类型
slideorder:99999,//排序号
slidelastupdatetime:'2016-08-16T16:25:00.003'//上一次修改时间
slidestate:0,//幻灯片状态 0 启用 1禁用
}
}
app的返回结果
{
state："0", //获取数据状态，0成功获取 2缺少参数 3错误的参数
imgroot:"G_UPLOAD_PATH", //图片根目录
"id":18, //最近一次活动表的id
"title":"五折大狂欢", //最近一次活动标题
"endtime":22588, //最近一次活动剩余秒数
"type":2, //类型 1 活动未开始 2活动进行中 3活动结束
data: //返回数据值
{
id:18,//活动id
slideid:1,//幻灯片id
slideimg:'slides/20160817/products_090542.jpg',//幻灯片地址
title:'五折大狂欢'//活动名称
}
}
