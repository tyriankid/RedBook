方法名称: adduseraddress (添加收货地址)
方法名称: addusergameaddress (添加游戏收货地址)（游戏）
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerruser?action=adduseraddress&sign=用户ID随机签名&addressid=地址ID
请求示例（游戏）
domain测试地址为：http://api.ewaywin.com
domain/ilerruser?action=addusergameaddress&sign=用户ID随机签名&gameaddressid=地址ID
参数说明
参数名称 类型 是否必须字段 说明
sign STRING true 用户ID随机签名，不接收未知的签名，微信不用传此参数
addressid STRING true 地址ID，更新地址是必须传此参数，添加地址不填
post array true POST提交大文本，格式 data:{sheng:省,shi:市,xian:县,jiedao:街道,youbian:邮编,shouhuoren:收货人,mobile:手机号,tell:座机,qq:QQ号}
参数说明（游戏）
参数名称 类型 是否必须字段 说明
sign STRING true 用户ID随机签名，不接收未知的签名，微信不用传此参数（游戏）
gameaddressid STRING true 地址ID，（更新地址必须此参数，添加不需要）（游戏）
post array true POST提交大文本，格式 data:{gameaddressid:游戏地址ID,gamename:游戏名称,fuwuqi:游戏服务器,daqu:大区,gamenum:游戏账号,shouhuoren:收货人,mobile:手机号}（游戏）
返回结果
{
state:"0", //获取添加收货地址，0添加/更新成功 //2未知的签名参数
}
（游戏）： {
state:"0", //获取添加收货地址，0添加/更新成功 //2未知的签名参数
}
