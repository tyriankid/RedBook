方法名称: article (获取文章列表)
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerrarticle?action=article&p=页码&cateid=所属分类id
参数说明
参数名称 类型 是否必须字段 说明
p STRING false 页码，默认为1
cateid STRING false 所属分类id，默认查全部
返回结果
{
state："0", //获取数据状态，0成功获取 1未获取到数据或数据已加载完毕
"count":12, //总条数，总数量
"pageCount":2, //总共页数
"nextPage":2, //下一页页码
imgroot:"G_UPLOAD_PATH", //图片根目录
data: //返回数据值
{
ordersn:'99999',//排序号
thumb:'article/20160816/products_042031.jpg',//图片地址
id:67,//文章唯一标识ID
title:'爽乐购哈哈哈',//文章标题
hit:0,//点击量
posttime:'2016-08-16T16:25:00.003'//提交时间
}
}
