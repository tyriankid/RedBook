方法名称: mysharelist (我的晒单)
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerrorder?action=mysharelist&sign=当前用户ID随机签名&p=页码
参数说明
参数名称 类型 是否必须字段 说明
sign STRING true 当前用户ID随机签名，不接收未知的签名,微信不用传此参数
p STRING false 页码，默认为1
