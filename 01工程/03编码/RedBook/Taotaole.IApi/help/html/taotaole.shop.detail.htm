方法名称: detail (获取商品详情)
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerrshop?action=detail&shopid=商品ID&productid=商品基本信息ID
参数说明
参数名称 类型 是否必须字段 说明
shopid STRING false 一元购商品ID。与productid参数二选一
productid STRING false 商品基本信息ID。与shopid参数二选一
uid STRING false 用户ID
返回结果
{
state:"0", //验证状态，0验证通过 2商品不存在
Q_showstate:'1', //揭晓状态 1进行中，2正在揭晓，3已揭晓
ismybuy:'0', //0代表未参入，1代表参入
ismywon:'0', //0是别人中奖，1是自己中奖
q_username:'张三', //中奖人昵称
q_img:'1.png', //中奖人头像
q_sumquantity:'10', //中奖人购买的总数
yId:'1', //1元购商品ID
Productid:'1', //商品基本信息ID
Title:'surface pro 4', //商品名称
...... //商品属性，具体以最终数据库为准(go_yiyuan、go_products)
}
