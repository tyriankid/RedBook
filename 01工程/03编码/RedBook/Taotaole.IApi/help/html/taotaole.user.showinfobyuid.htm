方法名称: visituser (查看某用户基本信息)
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerruser?action=visituser&sign=用户ID随机签名&otherid=其它用户的ID
参数说明
参数名称 类型 是否必须字段 说明
sign STRING true 用户ID随机签名，不接收未知的签名。微信不用传此参数
otherid STRING false 别人的用户ID
返回结果
{
state:"0", //状态，0验证通过 1用户不存在
imgroot:"G_UPLOAD_PATH" //图片根目录
username:"明明",//用户昵称
email:"123@163.com",//用户邮箱
qianming:"你好丑",//用户签名
touxiang:"123.jpg",//用户头像
luckyb:"0.00",//幸运币
money:"123.00",//金额
}
