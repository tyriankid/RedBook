方法名称: huifushaidan (发表商品晒单评论)
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerrshareorder?action=huifushaidan&sign=用户签名&shaidanid=晒单ID
参数说明
参数名称 类型 是否必须字段 说明
sign STRING true 用户ID随机签名，不接收未知的签名
post array true POST提交大文本，格式 data:{content:回复内容}
返回结果
{
state:"0" //状态，0回复成功 2未知的签名参数 3你发表次数过多哦 4你发表过于频繁 5发表失败
}
