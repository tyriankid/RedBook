方法名称: addressidlist (查看地址详情)
方法名称: gameaddressidlist (查看地址详情)（游戏）
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerruser?action=addressidlist&sign=用户ID随机签名

请求示例（游戏）
domain测试地址为：http://api.ewaywin.com
domain/ilerruser?action=gameaddressidlist&sign=用户ID随机签名
参数说明
参数名称 类型 是否必须字段 说明
sign STRING true 用户ID随机签名，不接收未知的签名，微信不用传此参数
addressid STRING true 地址唯一ID
