/ilerrshop?action=listinfo
方法名称: (获取商品列表)
请求示例
domain测试地址为：http://api.ewaywin.com
domain/ilerrshop?/action=listinfo&typeid=商品分类ID&order=排序ID&p=当前页码
参数说明
参数名称 类型 是否必须字段 说明
typeid STRING false 商品分类ID，默认为list,表示所有分类
order STRING false 排序ID，默认为10。10表示剩余人数升序、20表示人气、30表示参与人数倒序、40表示时间倒序、50表示价格倒序、60表示价格升序
p STRING false 当前页码，默认为1
返回结果
{
state："0", //获取数据状态，0成功获取 1未获取到数据或数据已加载完毕
count:"163", //总记录数
pageCount:"3", //总页码
nextPage:"3", //下一页码，为0时表示数据加载完毕
imgroot:"G_UPLOAD_PATH", //图片根目录
data:$shop //商品实体(仅公开部分属性)
{
id:'1',//商品ID
title:'surface pro 4',//商品名称
thumb:'1.jpg',//商品缩略图
qishu:'3',//商品期数
zongrenshu:'100',//总人数
shenyurenshu:'30',//剩余人数
canyurenshu:'70',//参与人数
}
}
