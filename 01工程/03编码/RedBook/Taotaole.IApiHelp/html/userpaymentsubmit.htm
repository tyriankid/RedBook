方法名称: userpaymentsubmit (用户提交付款)
请求示例
domain测试地址为：http://888.ewaywin.com
domain/?/mobile/APIorder/userpaymentsubmit/用户ID随机签名/商品ID/购买数量/支付方式/支付授权码/红包ID
参数说明
参数名称 类型 是否必须字段 说明
segment(4) STRING true 用户ID随机签名，不接收未知的签名
segment(5) STRING true 商品ID
segment(6) STRING true 购买数量
segment(7) STRING true 支付方式 0余额支付 1支付宝支付 2微信支付
segment(8) STRING true 支付授权码
segment(9) STRING false 红包ID,默认为0
返回结果
{
state:"0", //个人私有数据凭证状态，0验证通过 1缺少签名参数 2未知的签名参数 3缺少参数 4商品不存在 5商品已售罄 6商品已截止 7请不要重复提交 8用户登录超时 9余额不足
newshopid:"163" //最新一期的商品ID，当state为6时返回此参数。界面应提示： 该期已截止,是否参与最新期次？
APIsubmitcode:"ssg163" //支付授权码
}
