方法名称: userpayment (用户付款)
请求示例
domain测试地址为：http://888.ewaywin.com
domain/?/api/taotaolepay/userpayment/用户ID随机签名/商品ID/购买数量
参数说明
参数名称 类型 是否必须字段 说明
segment(4) STRING true 用户ID随机签名，不接收未知的签名
segment(5) STRING true 商品ID
segment(6) STRING true 购买数量
返回结果
{
state:"0", //个人私有数据凭证状态，0验证通过 1缺少签名参数 2未知的签名参数 3缺少参数 4商品不存在 5商品已售罄 6商品已过期
num："3", //实际购买数量
moenycount:"3", //实际待支付金额
APIsubmitcode:"163" //支付唯一码
}
